# LassoCMAQ

LassoCMAQ is an R Shiny application for simulating ozone and PM2.5 pollution levels based on customizable emission scenarios.

The app allows users to adjust emission factors for 17 administrative regions and 7 source categories, then visualize predicted air pollution levels across four representative months: January, April, July, and October.
